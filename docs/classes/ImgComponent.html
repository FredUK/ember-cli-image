ImgComponent Class

Uses
  • ImageLoaderMixin
Extends Ember.Component
Defined in: addon/components/img-component.js:4

img-component renders a stateful <img> element whose loading and error states can be observed, and whose class names are updated accordingly.

Instances of img-component can be created using the stateful-img Handlebars helper.

{{stateful-img src="img/image1.jpg" alt="Image" width=100 height=100}}
