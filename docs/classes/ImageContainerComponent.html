ImageContainerComponent Class

Uses
  • ImageStateMixin
Extends Ember.ContainerView
Defined in: addon/components/image-container-component.js:8

A container component with a stateful image as a child component. Class names are updated according to the image's state.

Instances of ImageContainerComponent can be created using the image-container Handlebars helper.

{{image-container src="img/image1.jpg" alt="Image" width=100 height=100}}
